Formatting

Tag < abbr > (Abbreviation) = Berfungsi untuk menampilkan singkatan berupa tooltip ketika ada mouse di atas text yang menggunakan tag ini, sebagai contoh : HTML

Tag < b > ( Bold ) = Berfungsi untuk membuat huruf bercetak tebal

Tag < address > = Berfungsi untuk mendefinisikan kontak dari penulis/pemilik dokumen atau artikel, sebagai contoh :

Pembuat dapat dihubungi di: suci.mariaa8898@gmail.com

Tag < center > = Berfungsi untuk membuat jajaran teks menjadi ditengah sebagai contoh :

Hallo

Tag < cite > (citation) = Berfungsi untuk membuat judul karya sebagai contoh, Referensi : www.3schools.com | www.tutorialspoint.com

Tag < del > = Berfungsi untuk membuat teks yang telah dihapus dari dokumen sehingga browser akan mencoret teks yang dihapus,dan tag < ins > berfungsi untuk menandai teks yang disisipkan. sebagai contoh : hal pertama yang akan dipelajari adalah PHP HTML

Tag < dfn > (definition) = Berfungsi untuk membuat sebuah istilah definisi, text yang menggunakan tag ini akan ditampilkan dengan huruf miring (italic) sebagai contoh : HTML adalah bahasa pemograman untuk membuat dan menyusun struktur halaman web

Tag < mark > = Berfungsi untuk membuat teks yang disorot / ditandai, sebagai contoh : saya sedang belaajar HTML

Tag < sub > (subscript text) = Berfungsi untuk membuat teks subskrip (seperti dalam penulisan zat Kimia), sebagai contoh : H2O

Tag < sup > (superscript text) = Berfungsi untuk membuat teks superscripted (seperti dalam penulisan akar kuadrat), sebagai contoh : 83 = 512

Tag < u > (underline) = Berfungsi untuk membuat karakter text dengan garis bawah, sebagai contoh : HTML atau Hypertext Markup Language adalah dasar dari semua halaman web di internet.

Tag < var > (variable) = Berfungsi untuk membuat sebuah variabel, sebagai contoh : Persamaan kecepatan adalah v = d / t

v = speed (kecepatan).

d = distance travelled (jarak).

t = time taken (waktu).
